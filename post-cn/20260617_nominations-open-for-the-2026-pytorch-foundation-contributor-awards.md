# 2026 PyTorch Foundation 贡献者奖项提名现已开放

**作者：PyTorch Foundation**
**2026年6月17日**

**2026 PyTorch Foundation 贡献者奖项**的提名现已开放！

这些奖项旨在表彰杰出个人，他们的贡献有助于强化 PyTorch Foundation 托管的项目，包括 PyTorch、vLLM、DeepSpeed、Ray、Helion 和 Safetensors，以及更广泛的社区。无论是技术创新、文档撰写、导师指导、倡导推广还是社区领导力，贡献者在推进我们使命中都发挥着至关重要的作用。

无论是通过代码、文档、指导还是社区领导力，贡献者都是我们社区蓬勃发展的核心。现在是你帮助我们庆祝他们成就的机会。

[立即提交提名](https://pytorch.org/contributor-awards/)

提名期于2026年6月17日开始，于2026年7月17日太平洋时间（PT）晚上11:59截止。

## 颁奖典礼

获奖者将获得一张[**2026年10月20日至21日在圣何塞举办的 PyTorch Conference**](https://events.linuxfoundation.org/pytorch-conference-north-america/)的免费门票。届时将举办颁奖典礼，对获奖者进行表彰。

## 你应该提名谁？

任何在 PyTorch Foundation 项目和社区中产生重要影响的人！我们欢迎并鼓励自我提名，以及来自各种背景、地区和角色的贡献者提名，包括：

* 开源开发者
* 文档撰写者
* 教育工作者和内容创作者
* 社区倡导者
* 项目贡献者和维护者
* Bug 报告者和修复者
* 长期贡献者和新兴新人

## 奖项类别

以下是你可以提名的类别列表：

* 杰出大使 — 在社区发展和吸引新用户加入生态系统方面表现卓越
* 超级英雄 — 在社区贡献的各个方面均表现卓越
* 生态系统冠军 — 在强化和发展更广泛项目生态系统方面表现卓越
* 知识增强者 — 在文档撰写、教程制作和知识分享方面表现卓越
* Bug 猎手 — 在发现、报告和解决 Bug 方面表现卓越
* 新人奖 — 在新贡献方面表现卓越
* 评审达人 — 在代码审查方面表现卓越
* 创新者 — 在创新功能、方法和核心架构贡献方面表现卓越
* 催化剂 — 在改善开发者工作流程、自动化、开发基础设施和 AI 辅助开发方面表现卓越

## 如何提交有力提名

通过提供贡献和影响的具体示例，帮助审查委员会了解被提名者值得认可的原因。

**具体说明**

描述被提名者的贡献以及他们如何带来了改变。避免笼统的陈述，专注于具体示例。

**突出影响**

_请考虑以下问题：_

* 他们的贡献如何改善了 PyTorch Foundation 项目或社区举措？
* 他们的工作是否帮助了用户、贡献者、维护者或开发者？
* 他们是否改善了文档、工具、性能、可靠性或可访问性？
* 他们是否通过指导、倡导、教育或活动支持了社区发展？

**提供支持证据**

请包含以下链接：

* GitHub 或 GitLab 的 Pull Request、Issue 或代码仓库
* 文档、教程或教育内容
* 博客文章、演讲、研讨会或演示
* 社区项目、活动或导师计划

**有力提名陈述示例：**

* _"主导了一次文档冲刺，改善了超过150篇教程，让新贡献者和用户更容易上手。"_
* _"解决了影响生产部署的关键 Bug，提升了下游用户项目的稳定性。"_
* _"组织了社区工作坊和培训课程，将 PyTorch Foundation 项目介绍给新的受众。"_
* _"通过成功的 Pull Request 和入职流程指导首次贡献者，帮助壮大了贡献者社区。"_

## 庆祝各种形式的贡献

我们欢迎来自社区各个部分的提名，跨越性别、地区、机构和贡献类型。贡献可包括倡导、教育、Bug 追踪、外联推广、翻译等。
